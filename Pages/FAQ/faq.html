  3. How do I play that game with Pokemon?

    Well, first of all, the game consists of three slots with sprites from a famous 8-bit game-franchise called Pokemon. The game is simple to play, you must get the same pokemon in all the slots. Watch out for the shiny ones too! and trust me, it's really hard to get the same pokemon in all 3 slots, the chances of getting a shiny are even vauge. So this game can get real tricky!
    [Click on any of the slots or refresh the web page to refresh the slots]

  4. What tools and Libraries did you use to make this site?

    I made this website with pure HTML, CSS & good ol' JavaScript! I used Sass or Sassy Style Sheets to make the styling workflow easier though. Animations and Transitions were made with GSAP or Gren Sock Animation Platform, Icons used are from FontAwesome.
